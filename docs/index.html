Architecture

How it works

01

Prove privately

A Circom circuit takes the issuer's private customer balances and proves their sum equals a public total, range-checking each one to block the Mt. Gox negative-liability trick.

02

Verify on Stellar

A Soroban contract checks the Groth16 proof with Stellar's native BLS12-381 host functions (Protocol 25/26), then confirms the attested reserve covers the proven liabilities.

03

Trust the badge

Anyone reads SOLVENT / INSOLVENT from the chain. A tampered total makes the pairing check fail, so a lie can never be recorded as a valid attestation.
